Layanan

Banyak teknologi yang bermunculan di era zaman sekarang, tentunya dengan kemajuan yang sangat pesat ini kita harus belajar cara mengikuti zaman yang serba moderen, dengan adanya artikel ini kita akan belajar bahasa pemerograman yang khusunya untuk pengembangan aplikasi, seperti membuat Aplikasi Desktop, Mobail dll, lalu apa sih bahasa pemrograman dan fungsi dari bahasa pemrograman ?

Bahasa Pemrograman atau sering diistilahkan dengan bahasa komputer, adalah sebuah instrukis standar untuk memerintah komputer. Bahsa pemrograman ini merupakan suatu himpunan dari sintaks dan semantik yang dipakai unruk mendefinisikan program komputer. Bahasa ini memungkinkan seorang programmer dapat menentukan secara pesis data mana yang akan di olah oleh komputer, dan bagaimana data ini akan disimpan/diteruskan, dan jenis langkah apa yang akan diambil dalam berbagai situasi secara persis.


Panduan Pembelajaran

1

Bahasa Perogam Java

Java adalah bahasa pemrograman yang bisa digunakan untuk keperluan para programmer untuk membuat sebuah program seperti: Java GUI Desktop, Java Mobail, webset dll

2

Photoshop

Photoshop adalah sebuah software yang digunakan untuk mengedit dan membuat sebuah disen, dan untuk membuat tampilan disen Desktop, mobail, webset dll.

3

Database

Database adalah untuk menyimpan data dari sebuah program seperti Desktop, mobail, dan webset dll, karna itu sebagai seorang programer harus bisa menggunakan database sebagai alat penyimpanan data.
